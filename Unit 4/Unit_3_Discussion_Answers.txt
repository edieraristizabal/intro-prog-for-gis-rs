1. Sample environment variables:
o cartographicCordinateSystem: Tools honor the specified cartographic
coordinate system.
o Workspace: Tools honor the specified directory as default location for
inputs and outputs.
o outputCoordinateSystem: Tools honor the specified cartographic
coordinate system for outputs

See: https://pro.arcgis.com/en/pro-app/arcpy/classes/env.htm

2. Required parameters of geoprocessing tools are values that required for
geoprocessing tools to function/run without errors. Optional parameters of geoprocessing
tools are values that are not required for geoprocessing tools to function/run without
errors; optional parameters add additional functionality to geoprocessing tools.

Check out Buffer syntax: https://pro.arcgis.com/en/pro-app/tool-reference/analysis/buffer.htm
